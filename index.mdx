## How It Works

AgenticPal uses a three-layer architecture for intelligent task execution:

1. **Agent Layer** — High-level orchestration using LLM for intent parsing and routing
2. **Tool Registry** — Pydantic-based validation and standardized execution
3. **Services Layer** — Direct Google API integration for Calendar, Gmail, and Tasks

<Note>
AgenticPal is currently in active development. While the CLI and API are functional, the web UI is under construction.
</Note>
